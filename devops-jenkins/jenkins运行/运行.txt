jenkins要用到宿主机的docker，通过 which docker 命令查询后，发现应这样挂载：-v /usr/bin/docker:/usr/bin/docker   -v /run/docker.sock:/var/run/docker.sock 




docker run   -u  root    -d    -p 8080:8080 -p 50000:50000  --name   jenkinsdemo -v /usr/bin/docker:/usr/bin/docker   -v /run/docker.sock:/var/run/docker.sock   jenkinsci/blueocean

/var/jenkins_home/updates/default.json：上面UpdateCenter.xml的内容（去除updateCenter.post();【以默认就有的格式粘贴】） && google ==> baidu
